# Aurora Browser

Кастомный современный браузер на Python, PySide6 и QtWebEngine.

## Запуск

Если PySide6 уже установлен:

```bash
python main.py
```

Или просто откройте файл `Запустить Aurora.bat` двойным кликом.

Если библиотек нет, установите одной командой:

```bash
pip install -r requirements.txt
```

- новая стартовая страница Aurora с поиском, быстрыми действиями, закладками и историей;
- виджеты времени, погоды и рабочих пространств на Dashboard;
- адресная строка с поиском и быстрым копированием адреса;
- выбор и добавление поисковых систем;
- компактная верхняя панель с собственными SVG-иконками;
- боковая навигация с иконками;
- внутренние страницы закладок, истории и загрузок;
- папки закладок и карточки быстрых сайтов;
- приватное окно;
- открытие локальных файлов и PDF через QtWebEngine;
- поиск по странице;
- просмотр исходного кода и DevTools;
- масштаб страницы;
- командная палитра `Ctrl+K`;
- закрепление вкладок;
- группы вкладок;
- спящие вкладки;
- вертикальные вкладки;
- режим чтения;
- скриншот страницы;
- пауза и продолжение активных загрузок;
- настройки;
- редактор интерфейса: высота панели, прозрачность, скругления, расстояния, масштаб интерфейса, анимации, расположение вкладок;

- пресеты Material 3: тёмная и светлая тема;
- импорт и экспорт темы.

## Горячие клавиши

- `Ctrl+T` - новая вкладка
- `Ctrl+W` - закрыть вкладку
- `Ctrl+K` - командная палитра
- `Ctrl+L` - адресная строка
- `Ctrl+F` - поиск на странице
- `Ctrl+D` - добавить закладку
- `Ctrl+Shift+G` - группа вкладки
- `Ctrl+Shift+S` - усыпить другие вкладки
- `Ctrl+Shift+V` - вертикальные вкладки
- `Ctrl+Shift+P` - приватное окно
- `Ctrl++`, `Ctrl+-`, `Ctrl+0` - масштаб
- `F11` - полный экран

Данные профиля хранятся в `user_data/settings.json`.
